About FinelPy
====================

The Finite Element Python Library (FinelPy) is an open source library for Finite Element Method (FEM) definitions and solvers.

It is a work in progress from the Laboratory of Topology Optimization and Multiphysics Analysis (`LTM <https://www.ltm.fem.unicamp.br/home>`_) from the School of Mechanical Engineering (also `FEM <https://www.fem.unicamp.br/>`_) from Universidade Estadual de Campinas, in Brazil (`UNICAMP <https://www.unicamp.br/>`_). The main contributor from LTM is Asst. Prof. Heitor Nigro Lopes.

The source code for FinelPy is available on `GitHub <https://github.com/Lopes941/finelpy>`_.

It is also used for the Postgraduate course Finite Element I - IM381.
