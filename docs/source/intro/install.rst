How to install
=======================

Pre-requisites
------------------

FinelPy requires the following components to run:

* C and C++ compiler
* Python >= 3.8
* Pybind11
* Eigen

It also requires the following Python libraries:

* numpy
* scipy
* matplotlib
* vtk


Installing with pip
---------------------

FinelPy is available on `PyPI <https://pypi.org/project/finelpy/>`_, and can be easily installed by running on a terminal:

.. code-block:: bash

    pip install finelpy

FinelPy is available on Windows, MacOS and Linux.


Custom Installation
---------------------

For custom properties, one should download the source code at `GitHub <https://github.com/Lopes941/finelpy>`_. You can install it using:

.. code-block:: bash

    pip install .

Custom properties are work in progress. This library will be fully compatible with PETSc and SLEPc.
